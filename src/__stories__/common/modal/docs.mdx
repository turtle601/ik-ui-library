# Modal

<br />

## Overview

`Modal`컴포넌트는 모달을 쉽게 구현할 수 있는 컴포넌트입니다. 모달을 열고 닫을 수 있는 기능을 제공하고, 반응형 스타일링도 적용 가능합니다.

## Usage

```tsx
function ProtalWrapper() {
  ...
}

<Modal.Provider>
  <Modal.Open modalContent={<ProtalWrapper>안녕</ProtalWrapper>}>
    <Center
      etcStyles={{
        width: '150px',
        height: '100px',
      }}
    >
      Modal 열기
    </Center>
  </Modal.Open>
  <Modal.Portal dom={document.querySelector('#modal') as HTMLElement} />
</Modal.Provider>
```

Modal 컴포넌트의 활용 예시는 **Example**을 통해 확인 가능합니다.

### Modal.Provider

- 설명: Modal.Provider는 모달 컨텍스트를 제공합니다. 이를 통해 Modal에 필요한 데이터를 컴포넌트끼리 연동할 수 있습니다.

### Props

```tsx
children: React.ReactNode;
```

### Modal.Open

- 설명: Modal.Open은 모달을 열기 위한 트리거 컴포넌트입니다.

### Props

```tsx
interface OpenProps extends ComponentPropsWithoutRef<'button'> {
  /** content에 들어갈 UI */
  modalContent: React.ReactNode;
  /** Modal.Open에 표시될 UI */
  children: React.ReactNode;
  /** modal을 여는 기능 외의 추가 기능 선언 */
  externalOnClick?: VoidFunction;
  /** 외부 스타일 */
  etcStyles?: CSSProperties;
}
```

### Modal.Close

- 설명: Modal.Close는 모달을 닫기 위한 트리거 컴포넌트입니다.

### Props

```tsx
interface CloseProps extends ComponentPropsWithoutRef<'button'> {
  /** Modal.Close에 표시될 UI */
  children: React.ReactNode;
  /** modal을 닫는 기능 외의 추가 기능 선언 */
  externalOnClick?: VoidFunction;
  /** 외부 스타일 */
  etcStyles?: CSSProperties;
}
```

### Modal.Portal

- 설명: Protal 컴포넌트는 React의 `createPortal`을 이용하여 DOM 트리 외부에 있는 요소를 UI로 렌더링 할 수 있다.
  내부적으로 useModal 훅을 사용하여 모달의 열림 상태와 컨텐츠를 관리합니다.

### Props

```tsx
interface PortalProps {
  /** Portal이 렌더링될 DOM 요소 */
  dom: HTMLElement;
}
```

### Modal.Content

- 설명: Content 컴포넌트는 모달의 열림 상태를 감지하여 해당 컨텐츠를 렌더링합니다.
  내부적으로는 `useModal`훅을 사용하여 모달이 열려있는지 여부와 모달 컨텐츠를 가져옵니다.
