# Container

<br />

## Overview

Container 컴포넌트를 사용하면 콘텐츠의 최대/최소 너비를 지정하고, 중앙 정렬을 손쉽게 구현할 수 있습니다.  
추가 스타일도 `etcStyles`를 통해 유연하게 적용할 수 있습니다.

<br />

## Usage

```tsx
<Container
  maxWidth="1200px"
  minWidth="300px"
  etcStyles={{
    padding: '20px',
    backgroundColor: 'lightgray',
  }}
>
  <Item />
</Container>
```

활용 예시는 **Example**을 통해 확인 가능합니다.

### Props

```
export interface ContainerProps {
  /** 시맨틱 태그 설정 (기본값: 'div') */
  as,
  /** 최대 너비 설정 */
  maxWidth?: CSSProperties['maxWidth'];
  /** 최소 너비 설정 */
  minWidth?: CSSProperties['minWidth'];
  /** 추가 스타일 설정 */
  etcStyles?: EtcStylesType;
  /** 자식 요소 */
  children: React.ReactNode;
  /** 태그 관련 dom 속성 */
  ...attribute
}
```
