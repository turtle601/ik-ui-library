# Center

<br />

## Overview

Center 컴포넌트를 사용하면 자식요소의 컴포넌트를 가운데 정렬할 수 있습니다.

<br />

## Usage

```tsx
<Center
  etcStyles={{
    width: '200px',
    height: '200px',
    border: '1px solid red',
  }}
>
  <Item />
</Center>
```

활용 예시는 **Example**을 통해 확인 가능합니다.

### Props

```tsx
export interface CenterProps {
  /** 시맨틱 태그 설정 (기본값: 'div') */
  as,
  /** Flex 방향 설정 */
  direction = 'row',
  /** 추가 스타일 설정 */
  etcStyles = {},
  /** 자식 요소 */
  children,
  /** 태그 관련 dom 속성 */
  ...attribute
}
```
